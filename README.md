a simple text generator that makes your text look fancy using unicode characters

## styles included

- bold
- italic  
- bold italic
- script (the fancy cursive one)
- bold script
- fraktur (gothic vibes)
- bold fraktur
- double-struck (the hollow letters)
- sans-serif
- sans-serif bold
- sans-serif italic
- monospace

## how to use

1. open `index.html` in your browser
2. type something
3. click copy on whatever style you like
4. paste it wherever

that's it. no installation, no setup, no npm packages with 500 dependencies

## tech

- vanilla js (no frameworks because we're not building a space shuttle)
- css that actually makes sense
- proper unicode handling with `Array.from()` so the characters don't break
- copy to clipboard that works on both modern and slightly-less-modern browsers
